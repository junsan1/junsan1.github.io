# StockSharp

StockSharp, often abbreviated as S#, is an [open](../o/open.md)-source [trading platform](../t/trading_platform.md) that provides a comprehensive set of tools for the development of [automated trading systems](../a/automated_trading_systems.md). It is designed to cater to a wide [range](../r/range.md) of traders, from beginners to professional algorithmic traders and financial institutions.

## Core Features and Components

### 1. Designer

S#.Designer is a visual strategy designer that allows traders to create [trading strategies](../t/trading_strategies.md) without writing code.

Key features:
- Drag-and-drop interface for strategy creation
- Large library of pre-built elements (indicators, trading functions, etc.)
- Visual [backtesting](../b/backtesting.md) and [optimization](../o/optimization.md) tools
- Real-time strategy testing
- [Export](../e/export.md) strategies to C# code

### 2. Hydra

A set of tools for collecting, storing, and managing [market](../m/market.md) data.

Key features:
- Support for [multiple](../m/multiple.md) data formats and sources
- Real-time and historical data collection
- Built-in data storage system
- Tools for data analysis and visualization
- API for custom data integrations

### 3. Shell

The main application that integrates all StockSharp components into a single trading terminal.

Key features:
- [Real-time market data](../r/real-time_market_data.md) display
- [Order management](../o/order_management_in_trading.md) and [execution](../e/execution.md)
- Portfolio tracking and [risk management](../r/risk_management.md)
- Strategy monitoring and control
- Customizable user interface

### 4. API

A comprehensive API that allows developers to create custom trading applications using C#.

Key features:
- Extensive documentation and code samples
- Support for [multiple](../m/multiple.md) [asset](../a/asset.md) classes ([stocks](../s/stock.md), [futures](../f/futures.md), [options](../o/options.md), etc.)
- Integration with numerous brokers and exchanges
- High-performance, multithreaded architecture
- Event-driven programming model

## Strategy Development

StockSharp provides [multiple](../m/multiple.md) ways to develop [trading strategies](../t/trading_strategies.md):

1. **Visual Designer**: For non-programmers, using S#.Designer
2. **C# Programming**: Using S#.API for custom strategy development
3. **Hybrid Approach**: Combining visual design with custom C# code

Features for strategy development include:
- [Backtesting](../b/backtesting.md) engine with detailed [performance analytics](../p/performance_analytics.md)
- Walk-forward and Monte Carlo testing
- [Optimization](../o/optimization.md) tools (including [genetic algorithms](../g/genetic_algorithms_in_trading.md))
- Real-time paper trading for strategy validation
